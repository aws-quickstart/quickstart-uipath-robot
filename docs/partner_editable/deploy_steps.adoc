NOTE: When you deploy UiPath Robot, choose at least two Availability Zones within your Region. Make sure that the number of Availability Zones matches the number you chose for this deployment. This Quick Start creates public subnets, private subnets, and https://docs.aws.amazon.com/vpc/latest/userguide/vpc-nat-gateway.html[NAT gateways^] to allow instances to download packages and software without exposing them to the internet.
